# Customer Order Analysis Using Python

A Python project that analyzes e-commerce customer orders to classify products, identify purchasing patterns, and generate business insights.

## Overview

This project demonstrates the use of Python's built-in data structures (lists, tuples, dictionaries, sets) and control structures (loops, conditionals) to process and analyze customer order data.

## Features

- **Store Customer Orders** - Using lists, tuples, and dictionaries
- **Classify Products** - Map products to categories (Electronics, Clothing, Home Essentials)
- **Analyze Spending** - Calculate totals and classify customers as high-value, moderate, or low-value buyers
- **Generate Insights** - Revenue per category, top customers, cross-category analysis
- **Set Operations** - Find multi-category shoppers and common purchasing patterns

## Files

| File | Description |
|------|-------------|
| `customer_order_analysis.py` | Main Python script with analysis logic |
| `report.md` | Comprehensive business report with insights |

## How to Run

```bash
python customer_order_analysis.py
```

## Sample Output

- Customer spending summary with classifications
- Revenue breakdown by category
- Top 3 highest-spending customers
- Multi-category shopper analysis

## Technologies Used

- Python 3
- Built-in data structures (lists, tuples, dicts, sets)
